1º LIGAR O SERVIDOR
    -> Procurar por Wampserver no iniciar, clicar nele, autorizar e esperar ficar verde.

2ª ACESSAR O BANCO DE DADEOS
    -> Abra o Google Chrome e acesse o link: http://localhost/phpmyadmin4.9.7/db_structure.php?server=1&db=sonnar

3º RODE O PROJETO
    -> Abra o console segurando as teclas CTRL+", ou vá em: Terminal > New Terminal
    -> No terminal aberto digite: npm run dev
    -> Espere aparecer a mensagem: "pronto para user!"
    -> Acesse o link: localhost:3030 para fazer os testes
